# Arxiv Chart Data Generation

This repository contains scripts and notes for generating chart-focused QA, reasoning, and dense-caption data from extracted arXiv figures.

Key files:

- `generation_edit2.md`: edition1 summary and edition2 task list.
- `problems_edit1.md`: pilot issue analysis.
- `scripts/`: edition1 scripts.
- `scripts_v2/`: edition2 scripts.

Large generated data, extracted figures, JSONL outputs, review assets, and local caches are intentionally excluded from git.
